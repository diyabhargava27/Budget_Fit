💰 BudgetFit – Expense Tracker

BudgetFit is a Full Stack Expense Tracker Web Application built using the MERN stack. It helps users manage their income and expenses efficiently with real-time data updates and a clean desktop-optimized interface.

🔗 Live Demo: https://budget-fit-frontend-qdsr.onrender.com/

🚀 Features

✅ Add Income & Expenses
✅ View Transaction History
✅ Real-time updates using REST APIs
✅ MongoDB database integration
✅ Clean and responsive Desktop UI
✅ Secure backend API handling

🛠 Tech Stack

Frontend: React.js

Backend: Node.js & Express.js

Database: MongoDB (MongoDB Atlas)

Tools Used:-

Git & GitHub : Render (Deployment)
